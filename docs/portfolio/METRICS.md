# 성공 지표 & 가설 검증

---

## 1. North Star Metric (장기)

**「주간 활성 배포 페이지 수 (Weekly Published Pages)」**

- 퀵페이지의 가치 = 페이지가 실제로 **배포되어 공유**되는 것
- 가입 수·매출보다 먼저 **배포 행동**을 본다

---

## 2. MVP 단계 핵심 지표

| 지표 | 정의 | 목표 (가설 검증) | 측정 |
|------|------|------------------|------|
| **첫 배포율** | 가입(또는 편집 시작) 후 24h 내 1회 이상 배포 | ≥ 30% | 배포 이벤트 / 편집 진입 |
| **편집→배포 전환** | `/create` 세션 중 배포 완료 비율 | ≥ 40% | 퍼널 |
| **배포→공유율** | 배포 후 7일 내 트래킹 링크 생성 또는 외부 referrer 유입 | ≥ 20% | tracking_events |
| **재방문 편집** | 동일 사용자 7일 내 2번째 페이지 생성·수정 | ≥ 15% | 대시보드 |
| **로그인 모달 전환** | 배포 클릭 → 모달 → 로그인 완료 | ≥ 50% | (구현 시 이벤트) |

---

## 3. 퍼널

```
랜딩 방문
  → 템플릿 선택
    → 편집 시작 (/create)
      → 배포 시도
        → [비로그인] 로그인 모달
          → 로그인 완료
            → 배포 성공
              → 트래킹 링크 생성
                → 외부 유입 (UTM)
```

**이탈 구간 우선 개선**:
1. 편집 시작 → 배포 (UX·로그인 마찰)
2. 배포 → 트래킹 링크 (가치 인지)
